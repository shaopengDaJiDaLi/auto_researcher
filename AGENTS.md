# AGENTS.md — Project-level rules for AI assistants

> For any AI assistant operating in this repository.

## Git Safety

1. Use the Git identity configured by the repository owner or by the current user.
2. Do not change repository visibility.
3. Do not delete the repository without explicit user authorization.
4. Do not force push to `main` without explicit per-operation authorization.

## Documentation

1. Keep the English and Chinese READMEs synchronized. Any user-facing change to `README.md` must be reflected in `docs/README_CN.md`, and vice versa.
2. Both READMEs must keep visible language switch links between English and Chinese.
